Description

An evolutionary algorithm that creates abstract interpretations of famous artworks. The system analyzes the original image's characteristics (edge density, color variety, brightness range) to determine an appropriate artistic style, then generates and evolves designs to match the source image.

Technical

The project uses p5.js for image processing and rendering. Key features include:

  • Image analysis for edge detection, color variety, and saliency mapping
  • Style selection based on image characteristics
  • Genetic algorithm for evolving designs
  • Multiple shape types (rectangles, circles) with style-appropriate parameters

Reflection

In reflection, I honestly liked the more pixel art-like look that I had early on, but these final results fit the assignment description more. Early on in the project I had chosen to use the statue of liberty as one of my images. I thought it was iconic in both shape and color, so it should stand out easily. I found the other two images later on after I had realized what my generator was good at. I think it is good at impressionism, with the colors and general positions matching, but the shapes abstracted and colors simplified. I chose to go with color, because my initial ideas were to make the outputs look similar to pixel art, which is possible in grayscale, but would have required higher fidelity outputs, with many more, smaller shapes. I liked that I found a way to accurately describe the general idea of the image with simple shapes, adding them on top of each other to create texture.
