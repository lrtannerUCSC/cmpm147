Alternate Worlds

Description

A procedurally generator that can create forests, with rivers, clearings, and houses, or dungeons with chests and passageways.

Technical

The world uses Perlin noise for river generation and cellular automata for terrain. The day/night cycle is controlled by a sine wave function, and houses emit light during nighttime. The dungeons are generated the same way, except that they generate paths between rooms and clickable chests.

Reflection

The part that gave me the most difficulty was creating the lookup table and understanding the grid of the tilemap to place the correct tiles. I also liked figuring out the code to create passageways. I am not very experienced with HTML so creating more buttons was also somewhat difficult
