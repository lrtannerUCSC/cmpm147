Whack-a-Mole: Infinite Capitalism

Artist Statement

Whack-a-Mole: Infinite Capitalism is a satirical take on clicker games and mindless consumerism, wrapped in the familiar mechanics of the classic arcade game. The project explores themes of exploitation, endless growth, and the absurdity of capitalism—where the player is incentivized to breed more moles just to whack them for profit.

The game starts simple: click moles, earn coins, buy upgrades. But as the player progresses, they unlock the ability to increase mole spawn rates and create more holes, turning the game into a grotesque parody of capitalist expansion. The minimalist black holes and white moles maintain focus on the core mechanic, while subtle red flashes provide satisfying feedback when a mole is whacked.

Technical Description

Built using p5.js, Whack-a-Mole: Infinite Capitalism leverages procedural generation and deterministic randomness to create an endless, replayable experience.

Core Mechanics

  • Procedural Hole Generation: Uses XXH32 hashing to determine hole placement
  • Mole Spawning Logic: Moles appear based on spawn rate with pop-up animations
  • Click Detection: Checks against deterministic seed for whack accuracy
  • Coin Economy: Whacked moles grant coins for upgrades

Upgrades

  • Auto-Whacker: Automatically whacks moles (cooldown decreases with upgrades)
  • More Holes: Increases number of active holes
  • More Moles: Boosts spawn rate and speed

Challenges & Solutions

  • Spam-Clicking Exploit: Left in as commentary on capitalist exploitation
  • Deterministic Randomness: Ensures fair mole appearances
  • Isometric Coordinate Confusion: Adjusted calculations for proper tile detection

How to Play

  • Click moles to earn coins
  • Buy upgrades to automate and expand
  • Profit endlessly—the moles never stop coming
  • (Note: No actual moles were harmed in the making of this game)

Development Notes

Original Concept: Asteroid Mining

The project began as an asteroid-mining game before pivoting when the black ellipses resembled mole holes. The asteroid concept was later revisited in another project.

Cookie Clicker Influence

Inspired by incremental games, the economy features:

  • Manual clicking (early-game effort)
  • Automation (Auto-Whacker)
  • Expansion (More Holes)

Future Expansions

  • Variable Mole Values (higher-value targets)
  • Visual Variety (golden/armored moles)
  • Capitalist Satire ("Mole Factories" upgrade)
